แนะนำชิปไมโครคอนโทรลเลอร์ของบริษัท Nordic Semiconductor#

▷ แนะนำชิป nRF SoC#

Nordic Semiconductor เป็นบริษัทผู้พัฒนาชิป SoC ซึ่งมีหลายตระกูล เช่น nRF51xx / nRF52xx / nRF53xx

ตาราง: ชิป SoC ของบริษัท Nordic Semiconductor พร้อมข้อมูลเชิงเปรียบเทียบ ในปีค.ศ. 2022 (Source: Nordic Semiconductor)

ตัวอย่างบอร์ดไมโครคอนโทรลเลอร์ที่ได้รับความนิยมและนิยมใช้งานสำหรับการสื่อสารไร้สายด้วยโพรโตคอล 2.4GHz Bluetooth / BLE เช่น

  • nRF51822 (ไม่แนะนำเลือกใช้งานสำหรับการออกแบบบอร์ดใหม่):
    • BBC Micro:bit V1.x
  • nRF52832 (ไม่แนะนำเลือกใช้งานสำหรับการออกแบบบอร์ดใหม่):
    • Sparkfun nRF52832 Breakout Board (retired)
    • Adafruit Feather nRF52832
  • nRF52840:
    • Nordic nRF52840-DK PCA10056
    • [SparkFun Pro nRF52840 Mini - Bluetooth Development Board](https://www.sparkfun.com/products/150250
    • Adafruit Feather nRF52840 Express
    • Adafruit Feather nRF52840 Sense
    • Adafruit ItsyBitsy nRF52840 Express
    • Seeed Studio XIAO nRF52840 / XIAO nRF52840 Sense
  • nRF52833:
    • BBC Micro:bit V2.x

Nordic nRF52840 Multi-Protocol SoC

  • CPU: 32-bit ARM Cortex-M4 with FPU, up to 64 MHz
  • On-chip Memory: 1MB Flash and 256KB SRAM
  • Wireless (2.4GHz)
    • Bluetooth 5: 2 Mbps, 1 Mbps, 500 kbps, and 125 kbps
    • IEEE 802.15.4-2006: 250 kbps
    • Proprietary 2.4 GHz: 2 Mbps, 1 Mbps
  • On-chip Crypto Engine: 128-bit AES/ECB/CCM/AAR co-processor
  • On-chip Interfaces
    • USB 2.0 full speed (12 Mbps) controller
    • QSPI (32MHz) interface
    • Type 2 Near Field Communication (NFC-A) Tag
    • Programmable peripheral interconnect (PPI)
    • 48 GPIO pins
    • EasyDMA data transfer between memory and peripherals
    • 8-channel, 12-bit, 200 ksps ADC
    • 4x 4-channel pulse width modulator (PWM) unit
    • Audio peripherals: I2S, digital microphone interface (PDM)
    • 5x 32-bit timer with counter mode
    • Up to 4x SPI master/3x SPI slave
    • Up to 2x I2C compatible 2-wire master/slave
    • 2x UART (CTS/RTS)
    • Quadrature decoder (QDEC)
    • 3x real-time counter (RTC)
