ตัวอย่างการรับส่งข้อมูลไร้สายระหว่างบอร์ด Micro:bit V2#

Keywords: BBC Micro:bit, Microsoft MakeCode, TypeScript, Wireless Data Communication

▷ บอร์ดไมโครบิตและการสื่อสารข้อมูลแบบไร้สาย#

บอร์ดไมโครคอนโทรลเลอร์ "ไมโครบิต" (Micro:bit) ทั้งรุ่นแรกและรุ่นที่สอง (v1 & v2) สามารถรับส่งข้อมูลสื่อสารไร้สาย (Wirless Data Communication) ด้วยคลื่นวิทยุความถี่ 2.4GHz เนื่องจากใช้ชิปของบริษัท Nordic Semiconductor รุ่น nRF51822 และ nRF52833 ตามลำดับ

ในบทความนี้จะนำเสนอตัวอย่างการเขียนโค้ดด้วย MakeCode Static TypeScript (STS) เพื่อรับส่งข้อมูลระหว่างบอร์ดไมโครบิต 2 บอร์ด และสำหรับการเขียนโค้ด ก็ทำได้สะดวก โดยใช้เว็บเบราว์เซอร์ Google Chrome เปิดใช้งาน MakeCode Editor for Micro:bit และสร้างโปรเจกต์ใหม่ (New Project)

MakeCode Editor for Micro:bit รองรับการเขียนโค้ดแบบต่อบล็อก (Block-based Coding) และเขียนโค้ดด้วยภาษา JavaScript (Static TypeScript) หรือ Python อีกทั้งยังสามารถจำลองการทำงานของโค้ดได้ (MakeCode Micro:bit Simulator) โดยยังไม่จำเป็นต้องใช้งานบอร์ดทดลองจริง

▷ การใช้งาน Multi-Board Editor#

เนื่องจากจะต้องใช้บอร์ดไมโครบิต 2 บอร์ด และทำงานพร้อม ๆ กัน ดังนั้นถ้าจะจำลองการทำงานของบอร์ดทั้งสอง จะต้องเปิดใช้งาน Multi-Board Editor

ผู้ใช้สามารถสร้างโปรเจกต์แยกกันระหว่าง 2 บอร์ดที่จะใช้งาน แล้วเปิดโปรเจกต์ใน Editor ทั้งซ้ายมือ และขวามือ
