Course Syllabus for Microprocessors & Embedded Computer Systems#

  • Course ID: 010113027
  • Academic Year / Semester: 2025 / 2
  • Course Type: Lecture (2 hours / week) & Hands-On Lab (2 hours / week)
  • Course Section Instructor: RSP
  • Target Audience: Undergraduate Electrical Engineering Students, ECE / KMUTNB

Course Description#

Introduction to microprocessors; structure and architecture of microprocessors; assembly language; memory and peripheral interfaces; high-level programming languages; embedded computer applications and testing; concurrent event response; interrupts; real-time control; sensor and actuator interfaces; applications in control and measurement systems; operating systems in embedded systems.

List of Topics#

  • Introduction Embedded Systems
  • The AVR Microcontroller: History and Features
  • Software Tools for AVR-based Software Development
  • AVR Architecture, Memory Map
  • AVR Assembly Language Programming
  • Branch, Call, and Time Delay Loop
  • Arithmetic, Logic Instructions, and Programs
  • AVR Programming in C, Hex File, and Flash Loaders
  • AVR I/O Ports, External Interrupts, Pin Change Interrupts
  • Polling vs. Interrupt-driven Programming for AVR
  • Low-power Programming for AVR
  • AVR Watchdog Timer (WDT)
  • AVR USART in Asynchronous Mode
  • AVR ADC & Analog Comparator
  • AVR Timer & PWM Programming
  • AVR SPI, I2C
  • Hardware Interfacing (e.g. WS2812B RGB LED)
  • Mini Project Presentation
